# MIMO-Nonlinear-Adaptive-Control
Development, implementation and virtual testing of a MIMO nonlinear adaptive control scheme for a UAV in package delivery applications. 

# About this repo: 
- This project was developed for academic purposes at Politecnico di Milano. 
- "Presentation.pdf" provides an overview of the project and presents its main results. It is strongly advised to have a look at this presentation before exploring the code. 
- The "Code" folder presents the MATLAB and Simulink code that underlies the presentation. It is structured into 4 subfolders, following the presentation's structure. In each folder, a MATLAB livescript (.mlx) file develops the task at hand. The simulink models and auxiliary functions it relies on are also included in the corresponding "Models and Auxiliary Functions" folder.  Tasks 3 and 4 also contain a subfolder dealing with relevant subtask (PBMRAC Development and Trajectory Generation, respectively). It is strongly advised to respect the folder and file structure to avoid dependency issues.
